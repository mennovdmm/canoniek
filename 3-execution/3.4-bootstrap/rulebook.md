## How to update this file
- A new learning = one bullet: `- [YYYY-MM-DD] <rule> (re: <short trigger>)`.
- A contradicting older rule? Update it or strike it through — no duplicates.
- Only real, reusable lessons. No one-off facts.

- Never write a secret into the core or version control.
