# 3.4 — Bootstrap

The entry point for a new user — the skill that builds your framework *with* you. Run it in Claude Code after
cloning: it explains the shape, scans what you already have, interviews you through the
[identity slots](../../1-core/1.3-identity-placeholders.md), generates your first `core` from
[`1.2`](../../1-core/1.2-mindset.md), and points you at the next steps.

- The skill: [`SKILL.md`](SKILL.md) + [`rulebook.md`](rulebook.md). It asks questions; it never assumes a
  specific business.
