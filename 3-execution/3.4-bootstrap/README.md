## What this skill must do
The entry point for a new user. Run in Claude Code, it:
1. **Interviews** the user to fill in the identity placeholders ([1.3](../1-core/1.3-identity-placeholders.md)).
2. **Scans** what the user already has (existing notes, structure, tools) so it doesn't ask what it can infer.
3. **Generates** the user's first `core` (their instantiated [1.2 mindset](../1-core/1.2-mindset.md)).
4. **Points** them to the next steps: build a first skill, set up state, wire the learning loop.

> This is the "build your framework *with* you" experience. Keep it generic — it asks questions, it never
> assumes a specific business.
