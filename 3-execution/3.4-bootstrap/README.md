# 3.4 — Bootstrap

The entry point for a new user — the skill that builds your framework *with* you. Run it in Claude Code after
cloning: it explains the shape, scans what you already have, interviews you through the
[identity slots](../../1-core/1.3-identity-placeholders.md), **generates your core into the root `CLAUDE.md`**
(instantiating [`1.2`](../../1-core/1.2-mindset.md)) so Claude Code auto-loads it next session, and points you
at the next steps.

> **This page is the human explanation.** The **runnable** skill lives in the machine layer at
> [`.claude/skills/bootstrap/`](../../.claude/skills/bootstrap/SKILL.md) (`SKILL.md` + `rulebook.md`) — that's
> what Claude Code auto-discovers and runs. The numbered handbook (here) is the reference; `.claude/` is the
> loaded layer. It asks questions; it never assumes a specific business.
