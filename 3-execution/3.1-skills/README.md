## What this folder must contain
- **What a skill is:** an on-demand, repeatable way of working (a capability) — a `SKILL.md` (with a sharp
  auto-load `description`) + a `rulebook.md`. Status lives in the registry, not in the folder.
- **The format:** point to the skill template in [3.3](../3.3-templates/).
- **The rulebook mechanic:** every correction → a dated rulebook line → next run loads it → better.
- **One golden example:** a complete, fully anonymized skill (SKILL.md + rulebook.md) that shows the pattern
  without leaking any use-case content.
