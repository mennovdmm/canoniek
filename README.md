# Canon

**An open framework for giving AI the full context of your business — and the discipline to keep it learning.**

The problem was never the AI. It's the context. People feed AI a snippet and get a tidy answer that still
misses what actually matters for their specific situation — because the AI doesn't know the business, and
nobody trains it structurally. Canon is the *structure and discipline* for fixing that: a way to hand your
knowledge, files and way-of-working to AI so every output is built on your own expertise.

Canon ships the **schema, not the data**. The structure is universal (you clone it); the content is yours
(you fill it in). You bring this repo into Claude Code, and Claude Code builds *your* framework *with* you —
interview, fill in, grow it as you go.

> **Status:** work in progress. This is a methodology handbook you read in order and instantiate — not a
> software product to install.

---

## How to use this repo
1. Read it top to bottom — it's numbered like chapters.
2. Open it in Claude Code and run the **bootstrap interview** ([`3-execution/3.4-bootstrap/`](3-execution/3.4-bootstrap/)).
   It asks who you are and generates your first `core`.
3. Fill in your layers, build your first skill, let the learning loop run.
4. See [`SETUP.md`](SETUP.md) for the one hard rule: **your data lives outside this repo.**
